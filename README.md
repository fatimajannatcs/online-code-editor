# Online Code Editor 💻

A browser-based code editing application with real-time execution support.

## 🛠️ Tech Stack
- React.js
- JavaScript
- CSS

## ✨ Features
- Real-time code editing
- Code execution in the browser
- Clean, responsive UI

## 📦 How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
